Apa itu Github?

Github adalah sebuah platform untuk pengembangan perangkat lunak dan kontrol versi menggunakan Git.

Github mulai dikembangkan pada 17 Oktober 2007.

Kegunaan Github

  • Memudahkan kolaborasi project
  • Mencegah perubahan kode yang bisa merusak kode asli
  • Sebagai portofolio bagi developer

Cara Menggunakan Github

  • Pertama, buat akun terlebih dahulu, kemudian lakukan verifikasi akun.
  • Kedua, mengisi profil data.
  • Ketiga, mulai project baru dengan membuat repository baru "create new repository. Isi nama repository sesuai keinginanmu
  • Keempat, repo baru telah dibuat. repo tersebut akan menjadi repo utama dimana repo tersebut dapat kamu tambah, ubah, clone ke mana saja.
